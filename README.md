# Big Data/Analytics for Game-Based Learning (Adaptive Quiz)

Adaptive general-knowledge quiz for university learners using learning analytics + machine learning.

## What this project includes
- Descriptive analytics (difficulty vs accuracy/time/hints/attempts)
- Next-question correctness prediction (Logistic Regression baseline + model comparisons)
- Personalisation policy (difficulty/topic/support recommendations)
- Learner profiling (K-Means clustering)
- Streamlit demo app (playable)

## Folder structure
- notebooks/ : Colab notebooks (00–07)
- figures/   : saved plots
- report/    : exported tables + chapter docs
- app/       : Streamlit app + question bank
